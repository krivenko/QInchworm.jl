    τ_i::Keldysh.TimeGridPoint,
    τ_f::Keldysh.TimeGridPoint,
    val::Dict{Int64, Tuple{Int64, Matrix{ComplexF64}}}
)

Set the value of expansion.P corresponding to a given pair of imaginary time points $(\tau_f, \tau_i)$. This method is defined for the spline-interpolated imaginary-time propagators.

Parameters

  • expansion: Pseudo-particle expansion.
  • τ_i: Initial imaginary time $\tau_i$.
  • τ_f: Final imaginary time $\tau_f$.
  • val: Block matrix to set $P(\tau_f, \tau_i)$ to.
